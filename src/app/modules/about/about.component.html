About the Project

Welcome to the Todo List project! This project is a simple web application that allows you to manage your tasks and stay organized. It provides a user-friendly interface where you can create, update, and delete your todo items.

Key Features

  • Create new todo items with titles and descriptions.
  • Set the status of each todo item (Normal, Important, Completed).
  • Search for specific todo items by their titles.
  • Filter todo items based on their status (Normal, Important, Completed).

Technologies Used

This project is built using the Angular framework for the frontend development. It leverages the power of Angular components, services, and data binding to create a seamless user experience. The todo items are managed using a custom TodoService, which handles data retrieval, addition, and updates.

Get Started

To get started with the Todo List application, simply create an account or log in if you already have one. Once logged in, you can start managing your tasks and organizing your daily activities. Enjoy the simplicity and effectiveness of the Todo List!
